# 🧪 Meu Projeto com Pipeline CI/CD

## 📋 Sobre o Projeto

Este é um projeto de demonstração que implementa uma pipeline de CI/CD usando GitHub Actions.

## 🚀 Funcionalidades

- Calculadora com operações básicas (soma e divisão)
- Testes automatizados com Pytest
- Pipeline CI/CD configurada com GitHub Actions

## 📦 Como executar localmente

```bash
# Clone o repositório
git clone https://github.com/Wiwer/meu_projeto_pipeline.git

# Instale as dependências
pip install -r requirements.txt

# Execute os testes
python -m pytest test_calculadora.py -v
